# Changelog

## 0.2.0 — Sidebar & Flow Tracing

- **Sidebar TreeView** — dedicated Indexa panel in the activity bar with structured results
- **Execution flow tracing** — see call chains across files in a tree structure
- **Type-aware icons** — functions, components, services, hooks each have distinct icons
- **Click-to-navigate** — click any result to jump to the exact line in the file
- **Auto-start server** — no manual terminal commands needed
- **Example queries** — clickable examples grouped by intent (Understand, Trace, Find)
- **Improved error states** — clear messages with recovery guidance
- **Dependencies & Connections** — see how symbols relate to each other

## 0.1.0 — Initial Release

- Ask Indexa (Ctrl+Shift+I) — search your codebase
- Explain This — right-click selected code
- Show Flow — trace execution from selected symbol
- Find References — blast radius analysis
- Health Check — verify connection
- Reindex — trigger re-indexing from VS Code
